# Keyboard Shortcuts

## Session Management

| Shortcut | Action |
|----------|--------|
| `Cmd+T` | Open new session modal |
| `Cmd+W` | Close the active session (with confirmation) |

## Navigation

| Shortcut | Action |
|----------|--------|
| `Cmd+1` – `Cmd+9` | Switch to session by position in the sidebar |
| `Cmd+Shift+[` | Previous session |
| `Cmd+Shift+]` | Next session |

## Terminal

| Shortcut | Action |
|----------|--------|
| `Cmd+F` | Open terminal search |
| `Enter` | Next search match (when search is open) |
| `Shift+Enter` | Previous search match (when search is open) |
| `Escape` | Close search bar |
| `Cmd+click` | Open file path in VS Code (on clickable file paths) |
| `Shift+Enter` | Insert newline in Claude Code (when search is closed) |

## App

| Shortcut | Action |
|----------|--------|
| `Cmd+,` | Open Settings |
